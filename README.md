## ✨ ¿Qué es?

Una web sencilla, visual y **autocorregible** para trabajar la geometría básica en **4–5 sesiones**.
Pensada para bajo nivel académico: lenguaje claro, números fáciles, mucho apoyo visual y
**tareas para medir objetos reales del aula**. El progreso se guarda en el propio dispositivo.

> 🌐 **Web publicada:** <https://cite-steam-sanjoseba.github.io/geometria-desde-cero/>

## 📚 Contenidos

| Bloque | Tema |
|:---:|:---|
| **1** | Figuras planas y sus elementos (lados, vértices, base, altura, diagonal, radio, diámetro) |
| **2** | Perímetros y áreas (cuadrado, rectángulo, triángulo y círculo) |
| **3** | Figuras compuestas (descomponer, sumar áreas y restar huecos) |
| **4** | Teorema de Pitágoras (catetos, hipotenusa y diagonales) |
| **5** | Cuerpos, fórmula de Euler y volúmenes |
| **📏** | **Mídelo tú** — medir objetos reales del centro y comprobar el cálculo |

## 🎯 Características

- ✅ Ejercicios que **se corrigen solos** con pistas.
- 📏 Tareas de **medida física** con objetos del aula (pupitre, libro, lata, caja…).
- 🏁 **Reto final** de 10 preguntas mezcladas con puntuación.
- 💾 **Progreso guardado** en el navegador (con botón para reiniciar).
- 🖨️ **Ficha resumen imprimible** con fórmulas y datos del alumno/a.
- 📸 Aviso de *«Haz captura de pantalla para entregar»* al completar cada bloque.
- ♿ Diseño **responsive y accesible**, sin dependencias externas.

## 🚀 Cómo usarla

- **En clase:** abre la [web publicada](https://cite-steam-sanjoseba.github.io/geometria-desde-cero/) y compártela con el alumnado.
- **En local:** descarga el repositorio y abre `index.html` con un doble clic. No necesita servidor ni instalación.

## 🧑‍🏫 Para el profesorado

Dentro de la web, la sección **«Para el profe»** incluye una **rúbrica de evaluación** sobre 10
y una **guía de sesiones** (4–5 clases). El alumnado entrega mediante capturas de pantalla y la
ficha imprimible.

## 🛠️ Tecnología

HTML + CSS + JavaScript puro. Sin frameworks, sin backend, sin datos personales.
Publicable directamente en **GitHub Pages**.
